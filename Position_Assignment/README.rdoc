== README

1. Gå till applikationsroten i konsolen. Skriv rake db:migrate
2. Starta applikationen. Skriv rails s
3. Testa systemet genom att lägga till användare och applikationer.
4. För att få administratörsrättigheter måste du gå in i databasen och ange is_admin till 1 under tabellen Users.
5. När man är admin listas alla applikationer när man är inloggad och är under "http://localhost:3000/creators"
